# Skills

## Overview

Skills represent trained areas of expertise. They are tied to Attributes — when making a skill check, the player declares which listed Attribute they are rolling with based on their approach.

**Skill Roll = d20 + Core Attribute Bonus + Skill Bonus + Modifiers**

- Missing required tools or implements: **-1 or -2 penalty** to the roll
- Some skills have a speciality (Craft, Profession, Perform, Trade) — choose a specific field when purchasing

## Skill Bonus Caps

| | In-Class Skill | Out-of-Class Skill |
|---|---|---|
| Maximum Skill Bonus | +5 | +2 |
| Cost per +1 | 1 AP (flat) | +1: 4 AP / +2: additional 8 AP |
| Free from Class | +1 to 1 Class Skill | — |
