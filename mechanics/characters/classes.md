# Classes

## Overview

A Class defines a character's core identity — their primary attributes, the abilities they specialize in, the skills they train, and the features that make them distinctive.

### Class Rules
- Every character receives **1 Class for free** at character creation
- A **2nd Class** costs 50 AP; a **3rd Class** costs 100 AP
- A **4th Class** is not permitted
- Once you own a Class, its Advances are always available — you never need to declare which class you are "levelling in"
- Taking Advances outside your owned Classes costs more AP

### What You Receive When Purchasing a Class
- **+1 to one Class Attribute** (free)
- **+1 to one Class Attack Bonus** (free, choose from class attributes)
- **+1 to one Class Save** (free, choose from class attributes)
- **+1 to one Class Skill** (free)
- **2 Class Standard abilities** (chosen from the class ability list)
- **1 Class Special ability** (chosen from the class ability list)
- **1 Class Feature** (chosen from the class ability list)
- **Special Pool 1** for this class
- **1 free HP Advance** (amount depends on class HP tier)

## HP Tiers

Each class has an HP tier that determines how much HP is gained per HP Advance.

| Tier | HP per Advance | Classes |
|---|---|---|
| **Hard** | 5 HP | Soldier, Commander, Pugilist, Inquisitor, Reaver |
| **Tough** | 3 HP | Scoundrel, Scout, Archaeologist, Artificer, Naturalist |
| **Normal** | 2 HP | Orator, Occultist, Arcanist, Investigator, Friar |

One free HP Advance is granted when you purchase a Class.

## Class Features

Each class grants 1 free Class Feature on purchase, chosen from the class's list. Additional features can be purchased with AP.

| Feature Type | In-Class Cost |
|---|---|
| Standard Feature | 8 AP |
| Pinnacle Defence Feature | 20 AP *(requires +2 Save in the relevant attribute)* |

### Full Feature List

**Combat**
1. **Weapon Proficiency** — trained in a weapon group; no penalty for use
2. **Armour Proficiency** — trained in a type of armour; no Speed or attack penalties
3. **Shield Proficiency** — can use a shield without penalty
4. **Two-Weapon Fighting** — reduced penalties when wielding two weapons
5. **Combat Reflexes** — can make additional Opportunity Attacks per round
6. **Sneak Attack** — bonus damage when attacking an unaware or flanked enemy
7. **Mounted Combat** — bonuses to attack and Defence when fighting on horseback

**Resilience**
8. **Toughness** — bonus HP (flat amount)
9. **Second Wind** — recover HP once per Short Rest as a Minor Action
10. **Evasion** — on a successful Dex Defence, take no damage instead of reduced damage
11. **Danger Sense** — cannot be surprised; bonus to Initiative

**Skills**
12. **Skill Specialization** — choose one skill; roll twice and take the higher result
13. **Skill Expertise** — chosen Class skill cap increases from +5 to +6
14. **Tool Proficiency** — trained with a specific set of tools; no penalty for use

**Magic & Abilities**
15. **Spell Specialization** — choose an ability category; reduce its AP improvement cost by 1
16. **Ritualist** — can use Class Special abilities outside combat without spending from the Special Pool
17. **Implement Proficiency** — trained with a specific magical implement; bonus when using it

**Social & Utility**
18. **Bardic Knowledge** — gain +1 in all Academic skills regardless of Class

**Pinnacle Defence** *(20 AP each; requires +2 Save in the relevant attribute)*
19. **Pinnacle Strength Defence** — roll 2d20 and take the higher on Str Defence rolls
20. **Pinnacle Dexterity Defence** — roll 2d20 and take the higher on Dex Defence rolls
21. **Pinnacle Constitution Defence** — roll 2d20 and take the higher on Con Defence rolls
22. **Pinnacle Intelligence Defence** — roll 2d20 and take the higher on Int Defence rolls
23. **Pinnacle Wisdom Defence** — roll 2d20 and take the higher on Wis Defence rolls
24. **Pinnacle Charisma Defence** — roll 2d20 and take the higher on Cha Defence rolls
