# Core Mechanics

## Design Principles

- Characters advance from **Levels 1–10** only
- Growth is incremental — a Level 10 character is more specialised, not a demigod
- A Level 2 character is not twice as powerful as a Level 1 character
- **Advancement Points (AP)** are the universal currency for all character building

---

## Attributes

Six primary attributes, each ranging from **-5 to +5**, all starting at **0**.

| Attribute | Score | Attack Bonus | Defence Bonus | Defence Target |
|---|---|---|---|---|
| Strength (Str) | 0 | 0 | 0 | 10 |
| Dexterity (Dex) | 0 | 0 | 0 | 10 |
| Constitution (Con) | 0 | 0 | 0 | 10 |
| Intelligence (Int) | 0 | 0 | 0 | 10 |
| Wisdom (Wis) | 0 | 0 | 0 | 10 |
| Charisma (Cha) | 0 | 0 | 0 | 10 |

- No attribute may go below **-5** or above **+5**
- Attributes are modified by Class, Feats, and Level Ups
- Each attribute has a separate **Attack Bonus** and **Defence Bonus**, purchased independently with AP

### Defence Target
**Defence Target = 10 + Defence Bonus + Equipment Bonus + Circumstance Bonus + Effect Bonuses**

Used as a passive threshold — when an opponent rolls against you, they must meet or beat your Defence Target.

---

## Roll Mechanics

### Attack Roll
**Attack Roll = d20 + Core Bonus + Ability Bonus + Equipment Bonus + Circumstance Bonus + Effect Bonuses**

### Skill Roll
**Skill Roll = d20 + Core Attribute Bonus + Skill Bonus + Modifiers**

### Core Bonus Components
| Source | Range |
|---|---|
| Attribute Score | -5 to +5 |
| Class Bonus | 0 to +2 |
| Advancement Bonus | 0 to +2 |

### Additional Modifiers
| Source | Range |
|---|---|
| Ability Bonus | 0 to +2 |
| Equipment Bonus | 0 to +2 |
| Circumstance Bonus | -5 to +5 (GM determined) |
| Effect Bonus | Variable |

### Advantage
Rolling with **Advantage** means rolling 2d20 and taking the higher result.

---

## Attack and Defence

Any roll can be an attack against any attribute's Defence Target. The attribute used depends on the nature of the action:

- Shooting a bow → Dex Attack vs Dex Defence
- Casting a spell → Int Attack vs Wis Defence
- Lying to someone → Cha Attack vs Wis Defence
- Overpowering someone physically → Str Attack vs Str Defence

### Who Chooses the Defending Attribute
- By default, the **defender** chooses which of their attribute Defence Targets is used
- Some abilities force the **attacker** to choose — this makes the ability more expensive to build
- Defender choosing from multiple attributes = cheaper ability to build
- Attacker forcing the defending attribute = more expensive ability to build

---

## Hit Points (HP)

**Starting HP = 10 + Con modifier**

HP can be increased by purchasing HP Advances with AP. The HP gained per advance varies by class:

| HP Tier | Cost | HP Gained |
|---|---|---|
| Hard | 10 AP | +5 HP |
| Tough | 10 AP | +3 HP |
| Normal | 10 AP | +2 HP |

One free HP Advance is granted when you purchase a Class.

---

## Speed

- **Default Speed = 30' per round**
- One **Move Action** allows movement up to 30'
- Speed can be increased with AP or reduced by armour and effects

---

## Action Economy

Based on D&D 4th Edition.

### Each Turn
| Action | Description |
|---|---|
| **Standard Action** | Main action — attack, use a major ability |
| **Move Action** | Move up to your Speed, or downgrade from Standard |
| **Minor Action** | Small tasks — draw weapon, open door |
| **Free Actions** | Talking, dropping items; GM may limit |

**Downgrading:** A Standard Action can be used as a Move or Minor. A Move Action can be used as a Minor.

### Movement Options
| Action Used | Distance | Restrictions |
|---|---|---|
| Move Action | Up to 30' | — |
| Standard Action (as Move) | Up to 30' additional | — |
| Full Round (Sprint/Flee) | Up to 90' | No Minor Action, no Reactions |

### Outside Your Turn
| Action | Trigger |
|---|---|
| **Immediate Interrupt** | Triggers before an event resolves; counts against next turn's immediate action |
| **Immediate Reaction** | Triggers after an event resolves; same limit |
| **Opportunity Action** | Triggered when an enemy drops their guard (e.g. moving out of melee range) |
